#react16-lazy

### 项目概况
***
React16.6新特性lazy配合react-router v4体验，使用webpack4打包构建

### 项目主要技术结构

***
* react16.6.3
* webpack4
* react-router4

### 安装
***
在终端下操作

项目地址: (`git clone`)

```
git clone git@github.com:xpioneer/react16-lazy.git
```

安装node_modules依赖 `yarn`

```
yarn #in your command terminal
```
***


### 运行
启动开发服务: (http://localhost:8800)

```
yarn start
```

生产环境打包

```
yarn build #目前还未实现
```
